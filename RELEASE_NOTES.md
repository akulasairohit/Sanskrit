# Release v1.0.0 - Production Ready

## Features
- **Full Mobile Optimization**: Responsive Sidebar (Bottom Nav), Typography, and Layouts.
- **Production Deployment**: Dockerized setup with Nginx Proxy Manager for SSL.
- **Security**: Sanitized deployment scripts and improved project structure.
- **Sanskrit AI**: Chat, Dictionary, Grammar Analysis, and Resources.

## Setup
1.  Clone the repository.
2.  Set `GEMINI_API_KEY` and `VPS_PASSWORD` in your environment.
3.  Run `ops/scripts/automate_deploy.exp` to deploy.
